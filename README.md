# Implement microservice with NodeJS and NextJS

Build microservice from scratch with nodejs and nextjs

## This module include 3 folder:

- blog: Front end
- posts: posts service
- comments: comments service

## How to run ?

After clone this repo

1. Run posts service

```
cd posts
yarn dev
```

2. Run comments service

```
cd comments
yarn dev
```

3. Run client(UI)

```
cd client
yarn
yarn dev
```
